# 📧 Email Phishing Detection System

A real-time phishing detection system using Random Forest and TF-IDF 
vectorization, built as part of M.S. Cybersecurity Capstone (CYB 691) 
at Quinnipiac University.

Author: Jangam Rajkumar  
Course: M.S. Cybersecurity Capstone – CYB 691

---

## 🔍 Features
- Real-time Gmail monitoring via IMAP IDLE
- Machine learning classification (Random Forest)
- OCR for image-based phishing detection
- URL pattern analysis and heuristic scoring
- Desktop notifications and forensic HTML reports
- System tray dashboard UI

## 🛠️ Tech Stack
- Python, scikit-learn, TF-IDF
- Tesseract OCR
- IMAP / Gmail API
- Random Forest Classifier
